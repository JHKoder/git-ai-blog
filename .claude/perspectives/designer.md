# Designer Perspective Rules (토스 스타일)

- UX Writing: 친절하고 행동 중심의 한국어 사용 ("확인하기", "송금하기")
- Visual: 충분한 여백, rounded-3xl, Primary #0066FF, Pretendard 폰트
- Accessibility: WCAG AA, 터치 타겟 44px 이상, contrast 4.5:1 이상
- Feedback: 사용자가 "이게 왜 이렇게 됐지?"라고 느끼지 않게 직관적으로
- Review 포인트: 일관성, 심플리시티, 감정적 신뢰(Fintech Trust)
