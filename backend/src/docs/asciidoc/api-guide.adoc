== 개요

`git-ai-blog.kr` API 문서입니다.
모든 요청은 JWT 인증이 필요합니다 (`Authorization: Bearer <token>` 헤더).

=== 공통 응답 형식

[source,json]
----
{
  "success": true,
  "message": "OK",
  "data": { ... }
}
----

=== 오류 코드

|===
| HTTP 상태 | 설명

| 400 | 요청 파라미터 오류 (유효성 검사 실패)
| 403 | 인증 필요 (JWT 없음 또는 만료)
| 404 | 리소스를 찾을 수 없음
| 422 | 도메인 규칙 위반 (잘못된 상태 전이 등)
| 429 | AI 호출 한도 초과
| 503 | 외부 API(Hashnode, Claude 등) 응답 불가
|===
